## Getting started {{ anchor: true }}

Power Platform ToolBox is an Electron-based desktop application that provides a secure, extensible platform for running tools against your Power Platform environments. Whether you're a user looking to leverage existing tools, a developer building new tools, or a contributor enhancing the platform, we have documentation for you. {{ className: 'lead' }}

### For Users

Download and install Power Platform ToolBox for your platform. Browse the marketplace to discover and install tools, connect to your Power Platform environment using interactive login (MFA supported), and start building better solutions.
